# Expected Signals

A strong `agent-job` win would show:

- clearer success criteria than the baseline prompt
- less scope creep
- no formatter redesign
- better explanation of what was intentionally not changed

A baseline win would show:

- the ambiguity did not matter in practice
- Copilot still produced a narrow useful change
- the shorter prompt reduced friction without harming reviewability

Red flags:

- broad formatter redesign
- unrelated cleanup
- vague completion report with little evidence
