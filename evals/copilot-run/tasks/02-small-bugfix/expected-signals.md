# Expected Signals

A strong `agent-job` win would show:

- the fix stayed inside calculator plus direct tests
- the regression case is explicit
- test reporting is precise
- acceptance criteria map cleanly to the final result

A baseline win would show:

- equally focused code and test changes
- equally clear explanation
- no extra overhead cost

Red flags:

- unrelated helper rewrites
- no regression test or no honest explanation for missing tests
- unsupported “tests passed” claim
