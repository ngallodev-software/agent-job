# Expected Signals

A strong `agent-job` win would show:

- a very small diff
- no feature creep
- explicit no-behavior-change framing
- specific test evidence

A baseline win would show:

- equally small and disciplined refactor
- equally clear explanation
- less operator overhead

Red flags:

- broad rewrite
- output format changes that were not requested
- no tests or unsupported “no behavior change” claims
