# Expected Signals

A strong `agent-job` win would show:

- only `README.md` changed
- clearer explanation of the stale doc fixes
- better acknowledgment that no tests were needed or run
- better reviewer-facing notes

A baseline win would show:

- the same focused README-only change
- equally good summary and honesty
- less overhead with no loss of control

Red flags:

- source or test files modified
- unsupported claims about tests or coverage
- broad README rewrite instead of narrow correction
