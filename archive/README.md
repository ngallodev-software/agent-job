# Archive

This directory holds historical or non-canonical material that is intentionally kept out of the active repo surface.

Current archive contents:

- `future-plans/`: control-plane, queue, UI, and planning work that is not part of the current `agent-job` product path
- `deprecated/`: retired or superseded scripts and notes
- `tests/`: tests that only validate archived materials

Archive rules:

- do not treat archived files as the source of truth for current implementation work
- do not point new docs or tests at archive content unless the task is explicitly historical
- prefer active repo surfaces such as `agent-job/`, `evals/`, `README.md`, and current tests
